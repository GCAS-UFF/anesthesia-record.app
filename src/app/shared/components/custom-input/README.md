# CustomInputComponent

Input customizado reutilizável com ícone, label e integração com Reactive Forms.

## Inputs
- `label`: string (label do campo)
- `type`: 'text' | 'password' (tipo do input)
- `icon`: string (nome do ícone Ionic)
- `placeholder`: string
- `required`: boolean

## Exemplo de uso
```html
<app-custom-input label="Usuário" icon="person" required></app-custom-input>
```
