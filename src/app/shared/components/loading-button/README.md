# LoadingButtonComponent

Botão reutilizável com loading (spinner) e desabilitação automática.

## Inputs
- `loading`: boolean
- `disabled`: boolean
- `expand`: 'block' | 'full' | 'default'
- `color`: string
- `type`: 'button' | 'submit'

## Exemplo de uso
```html
<app-loading-button [loading]="true">Entrar</app-loading-button>
```
