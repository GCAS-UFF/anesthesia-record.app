# HeaderInstitucionalComponent

Componente reutilizável para exibir o header institucional (logo + nome do sistema).

## Inputs
- `logo`: string (opcional, caminho do logo)
- `title`: string (nome do sistema)

## Exemplo de uso
```html
<app-header-institucional [logo]="'assets/logo.png'" title="Ficha Anestésica"></app-header-institucional>
```
